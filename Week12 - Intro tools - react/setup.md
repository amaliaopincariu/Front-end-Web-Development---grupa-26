# Setup

## Sa va instalati Node.js

1. Intrati pe https://nodejs.org/en/
2. Download la versiunea 16.16.0 LTS
3. Urmati pasi de instalare - next next
4. Deschideti un terminal si rulati `node -v` ( va afisa versiune de la node)
